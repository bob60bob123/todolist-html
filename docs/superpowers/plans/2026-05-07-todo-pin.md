```javascript
function togglePin(id) {
    const todo = todos.find(t => t.id === id);
    if (todo) {
        todo.pinned = !todo.pinned;
        saveTodos();
        renderTodos();
        playSound('add');
```

```javascript
        // Apply pin sorting for 'all' and 'active' filters
        if (currentFilter === 'all' || currentFilter === 'active') {
            const pinnedTodos = f.filter(t => t.pinned);
            const unpinnedTodos = f.filter(t => !t.pinned);
            pinnedTodos.sort((a, b) => new Date(b.createTime) - new Date(a.createTime));
            unpinnedTodos.sort((a, b) => new Date(b.createTime) - new Date(a.createTime));
            f = [...pinnedTodos, ...unpinnedTodos];
        }
```

### Task 9: 修改 addTodo 设置默认 pinned: false

**Files:**
- Modify: `ToDoList.html:2211`

- [ ] **Step 1: 确保 addTodo 中 todo 对象包含 pinned 字段**

当前代码：
```javascript
todos.unshift({ id: Date.now(), text, priority, category, completed: false, createTime: new Date().toISOString() });
```

修改为：
```javascript
todos.unshift({ id: Date.now(), text, priority, category, completed: false, createTime: new Date().toISOString(), pinned: false });
```

注意：此修改为显式添加，虽然 JavaScript 中对象字面量可以省略 `pinned` 字段（初始化时为 undefined），但显式设置为 false 是更好的实践，便于后续代码正确读取
